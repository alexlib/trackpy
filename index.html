A Pure python implementation of Crocker-Grier¶

Release:0.2.0
date:October 22, 2012

The Crocker-Grier algorithm is a method of tracking features in a series of images from frame to frame. The core of the algorithm is to choose the frame-to-frame linking that globally minimizes the sum of the squared displacements.

trackpy is a simple and extendable implementation of Crocker-Grier suitable for tracking a few hundred to a few thousand features per frame. This implementation has been shown to consume an unreasonable amount of memory when tracking >20k features. For large data sets, see the c++ implementation of Crocker-Grier.
