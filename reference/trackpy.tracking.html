class trackpy.tracking.Point¶

Bases: object

Base class for point (features) used in tracking. This class contains all of the general stuff for interacting with Track objects.

Note

To be used for tracking this class must be sub-classed to provide a distance() function. Child classes MUST call Point.__init__(). (See PointND for example. )

trackpy.tracking.link(levels, search_range, hash_generator, memory=0, track_cls=<class 'trackpy.tracking.Track'>)¶
Parameters:
  • levels (Iterable of iterables) – Nested iterables of Point objects
  • search_range – the maximum distance features can move between frames
  • hash_generator – a callable that will return a new empty object that supports add_particle() and get_region()
  • memory – the maximum number of frames that a feature can skip along a track
  • track_cls (Track) – The class to use for the returned track objects

Generic version of linking algorithm, should work for any dimension. All information about dimensionality and the metric are encapsulated in the hash_table and Point objects.
